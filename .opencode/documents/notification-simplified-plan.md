```
A1 → A2 → A3 → A4 → B1 → B2 → B3 → B4 → C1 → C2
```

- A 阶段（后台）可以先完成，前端可以独立简化

## 检查点

1. **后台轮询服务启动后**：验证能正确检测数据库变化
2. **WebSocket 推送验证**：前端能收到通知消息
3. **Toast 显示验证**：收到通知后正确显示
4. **详情弹窗验证**：点击后能获取并显示详情
5. **已读标记验证**：关闭弹窗后状态正确更新
