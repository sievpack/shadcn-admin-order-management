# 代码注释规范（中文）
## 注释语言
- 所有 Python 代码的注释必须使用中文
## 函数级注释
- 每个函数上方必须有中文说明：
- 函数是做什么的
- 参数含义和类型
- 返回值含义
## 行内注释
- 对复杂逻辑行，在右侧或上方添加中文解释，说明「为什么」而不是「做了什么」
## 检查步骤
1. 扫描所有函数定义
2. 确认是否存在中文注释
3. 如缺失，自动补齐
4. 确保注释语言为中文
