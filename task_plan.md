# 打印功能实现计划

## 目标
在shadcn-admin项目中实现一个统一的打印接口，支持多种打印场景：二维码、加工单、送货单、样品单、采购单、合同等。

## 约束条件
- 不要太复杂
- 实现统一接口
- 保持与现有代码风格一致

## 项目结构分析
- 使用React + TypeScript + Vite构建
- 状态管理使用Zustand
- 路由使用TanStack Router
- UI组件基于Shadcn UI
- 功能按模块组织在src/features下
- 工具函数位于src/lib
- 服务位于src/services
